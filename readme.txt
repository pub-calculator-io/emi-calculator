[https://www.calculator.io/emi-calculator/](https://www.calculator.io/emi-calculator/)

== Usage ==

Add the EMI Calculator shortcode to your page, post or sidebar:

`[ci_emi_calculator]`

Add the following code to your template where you would like the EMI Calculator to appear:

`<?php display_ci_emi_calculator(); ?>`

1. Upload the EMI Calculator /ci_emi_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the EMI Calculator plugin through the "Plugins" menu in WordPress.
